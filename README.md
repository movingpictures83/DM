PluMA plugin to run CCD (Richardson, 1996)

Input is a TXT file of tab-delimited keyword-value pairs:
data_dir: Input file/directory
inputs: list
outputs: list
useGES: True or False
trueInputs: list
alphaPC: float
verbose: True or False
gesDiscount: integer
minDiscount: integer



Output is an image in PNG format.
